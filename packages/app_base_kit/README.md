# app_base_kit

A collection of essential utilities and base classes for Flutter applications.

## Installation

Add the dependency in `pubspec.yaml`:

```yaml 
dependencies:
  app_base_kit: ^1.0.0
```

Then run:

``` bash
flutter pub get
```
