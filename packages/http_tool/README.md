# http_tool

A network request library based on Dio, featuring functions such as request retry, logging,
interceptors, and business error handling.

## Installation

Add the dependency in `pubspec.yaml`:

```yaml 
dependencies:
  http_tool: ^1.0.0
```

Then run:

``` bash
flutter pub get
```
