# app_ui

A collection of reusable UI components and widgets for Flutter applications.

## Installation

Add the dependency in `pubspec.yaml`:

```yaml 
dependencies:
  app_ui: ^1.0.0
```

Then run:

``` bash
flutter pub get
```
