# Danmaku Widget

A Flutter widget for displaying animated danmaku (bullet comments) with customizable styles and
effects.

## Installation

Add the dependency in `pubspec.yaml`:

```yaml 
dependencies:
  danmaku_widget: ^1.0.0
```

Then run:

``` bash
flutter pub get
```
